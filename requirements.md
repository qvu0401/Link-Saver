# Project:

Build a **Link Saver** web app — users paste a URL, the app saves it with a title, tag, and a “read later / read” status. 

## User Stories:

1. **Student**

As a student, I want to save links with tags like "math" or "CSE 20" so that I 
can find all resources for a specific class in one place when I'm studying for 
an exam.

As a student, I want the title of a saved link to be automatically 
fetched from the webpage so that I don't have to manually name each link 
and can save articles quickly without interrupting my workflow.

As a student, I want to sort my saved links by the date they were saved so 
that I can quickly find recently saved links without searching through my 
entire list.

2. **Online Shopper**

As an online shopper, I want to save product links and mark them as "read later" 
so that I can make a better purchasing decision at the end of the week instead 
of impulse buying things immediately.

3. **Researcher**

As a researcher, I want to tag saved links with a topic so that I can filter
by tag and quickly find all resources related to a specific subject without
scrolling through my entire list/ remembering the title of the article.
